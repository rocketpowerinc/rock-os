Mission modules

Built For Local Resilience

Local First

Runs from your own machine and serves your docs across the local network without needing a cloud control plane.

Offline Ready

Markdown, local assets, media folders, and bootstrap notes stay useful when the internet is unavailable.

Cross Platform

Linux gets the full ISO path, while Windows, macOS, Android, and iOS get practical hardening and setup playbooks.

Private Knowledge

Git-friendly documentation with optional encrypted private folders for sensitive notes that should not leak in public.
