# Markdown Callouts

Use blockquotes that start with `[!TYPE]`.

> [!NOTE]
> This is a general note for extra context.

> [!INFO]
> This is informational text for something useful but not urgent.

> [!TIP]
> This is a helpful tip that makes a task easier.

> [!SUCCESS]
> This is a success message for completed or verified steps.

> [!WARNING]
> Do this before rebooting.

> [!DANGER]
> This is a dangerous step that can break something if done carelessly.

> [!ERROR]
> This is an error callout for something that failed.

> [!QUESTION]
> This is a question callout for things that still need a decision.

## With Markdown Inside

> [!TIP]
> You can still use **bold text**, `inline code`, and links inside callouts.
>
> - First item
> - Second item
