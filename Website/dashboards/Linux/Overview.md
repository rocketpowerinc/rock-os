# Linux Dashboard

This dashboard is the Linux command center for Rock-OS.

Use it for Linux maintenance notes, bootstrap reminders, distro-specific
commands, recovery steps, package manager references, and the kind of setup
breadcrumbs you only appreciate after forgetting them at 1 AM.

## Good Uses

- Package update and repair notes.
- Distro-specific install reminders.
- Service, log, and monitoring references.
- Backup and restore checklists.
- Offline installer and ISO notes.
- Recovery commands that should be easy to find before panic typing begins.

## Dashboard Notes

Dashboards are public local content. They are not encrypted with `git-crypt`.
Put private notes under Profiles instead.
