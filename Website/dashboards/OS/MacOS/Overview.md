# MacOS Dashboard

This dashboard is the MacOS corner of Rock-OS.

Use it for Mac setup notes, package workflows, backups, local sync, recovery
steps, and the practical details that keep a Mac useful without depending on a
cloud account for every tiny thing.

## Good Uses

- Homebrew and CLI tool notes.
- Backup and restore checklists.
- Local network file sharing reminders.
- Offline installer inventories.
- Privacy and hardening steps.
- Recovery notes for when macOS calmly says "something went wrong" and refuses
  to elaborate.

## Practical MacOS Notes

MacOS can be a very capable local workstation when the basics are documented:
package setup, data locations, backups, local network access, and recovery
steps.

Keep the notes boring in the best way:

- What needs to be installed.
- Where important data lives.
- How backups are made and restored.
- What to do after a fresh install.

## Dashboard Notes

Dashboards are public local content. They are not encrypted with `git-crypt`.
Put private notes under Profiles instead.
