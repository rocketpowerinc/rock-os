# Windows Dashboard

This dashboard is the Windows command center for Rock-OS.

Use it for Windows-specific maintenance notes, setup reminders, update
playbooks, recovery steps, hardening checklists, and anything else that should
be easy to reach from the local intranet.

## Good Uses

- Windows update and package management notes.
- Backup and restore checklists.
- Offline installer inventories.
- Hardening steps that should be repeated on new machines.
- Recovery notes for the day Windows decides it needs attention at the worst
  possible moment.

## Dashboard Notes

Dashboards are public local content. They are not encrypted with `git-crypt`.
Put private notes under Profiles instead.
