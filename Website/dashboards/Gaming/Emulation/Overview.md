# Emulation Dashboard

This dashboard is a local command center for emulation notes, emulator setup,
BIOS reminders, controller mapping, ROM organization, save management, and the
small details that keep a retro setup from turning into a drawer full of
mystery cables and regret.

## Good Uses

- Emulator install and configuration notes.
- Controller mapping references.
- BIOS and firmware checklists.
- Save-state and memory-card backup reminders.
- Per-system setup notes for consoles, handhelds, and arcade platforms.
- Troubleshooting steps for video, audio, shaders, and input.

## Dashboard Notes

Dashboards are public local content. They are not encrypted with `git-crypt`.
Put private notes under Profiles instead.
