# iOS Dashboard

This dashboard is the iOS corner of Rock-OS.

Use it for iPhone and iPad setup notes, offline app planning, local file sync,
backup habits, recovery steps, and the mobile workflows that still matter when
the cloud has wandered off to find itself.

## Good Uses

- iPhone and iPad setup notes.
- Offline app and media checklists.
- Local file sharing and sync reminders.
- Backup and restore workflows.
- Privacy and hardening notes.
- Recovery steps for when the tiny rectangle becomes the command center.

## Practical iOS Notes

iOS devices can be excellent local clients during an outage: portable, familiar,
battery powered, and easy for non-technical users to operate. The trick is
documenting the setup before stress enters the chat.

Keep these notes focused on what someone can repeat:

- Which apps are needed offline.
- How files get onto the device.
- Where backups live.
- What to check after a restore.

## Dashboard Notes

Dashboards are public local content. They are not encrypted with `git-crypt`.
Put private notes under Profiles instead.
